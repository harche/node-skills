# Presentations

## Prerequisites

Check if LibreOffice is installed (`soffice --version`). If not, strongly recommend the user install it before proceeding — it is important for automated verification and helps in generating better presentations.

```bash
# macOS
brew install --cask libreoffice

# Fedora/RHEL
sudo dnf install libreoffice-impress
```

## Red Hat Template

Use the Red Hat slide template as the base for all presentations:

```
./references/templates/rh-slide-template.pptx
```

Pass this template to the document skill so all presentations follow Red Hat branding.
